## Overview and Project Goals
The [goal](https://frontend.turing.edu/projects/module-3/rancid-tomatillos-v3.html) of this group project that was made for Mod 3 at [Turing School of Software and Design](https://turing.io/) is to use React in order to create an application for users to have access to information about a selection of movies.
